QJSON.

JSON superset with exact numerics, pattern matching, and SQL storage.

pip install qjson npm install qjson

What it adds

{
  price: 67432.50M,           // BigDecimal -- exact base-10
  count: 9007199254740993N,   // BigInt -- beyond 2^53
  query: ?X,                  // unbound -- matches anything
  key: 0jSGVsbG8,             // blob (JS64 binary)
  /* nested /* block */ comments */
}

// Complex keys -- any value can be a key
{42: "answer", [1, 2]: "pair"}

// Set shorthand -- sugar for {a: true, b: true, ...}
{alice, bob, carol}

// Datalog facts -- sets of tuples
{[alice, bob], [bob, carol]}

Valid JSON is valid QJSON. No collisions.

Features

Exact arithmetic

17 SQL functions (add, mul, pow, sqrt, sin, ...) backed by libbf. No floating-point surprises.

Interval projection

[lo, str, hi] brackets every value. Indexed range queries with exact fallback.

Constraint solver

Store 3 of 4 values, leave one as ?. The solver fills it in via leaf folding.

Transitive closure

qjson_closure() computes graph reachability via WITH RECURSIVE.

Complex keys + sets

Any QJSON value as object key. {a, b, c} set shorthand for Datalog predicates.

Encryption at rest

SQLCipher page-level encryption. WASM + OPFS for browser. Encrypted volumes for PostgreSQL.
